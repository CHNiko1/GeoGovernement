    Accessibility Statement

    The Georgian Government is committed to ensuring digital accessibility for all citizens, including those with disabilities.

    Our Commitment

    We are dedicated to providing a website that is accessible to the widest possible audience, regardless of ability or circumstance. This website aims to comply with World Wide Web Consortium (W3C) Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

    Accessibility Features

    • Keyboard Navigation: All functionality is accessible via keyboard alone. Press Tab to navigate and Enter to activate.
    • Screen Reader Support: Our website is designed to work with screen readers including NVDA, JAWS, and VoiceOver.
    • Color Contrast: Text has sufficient contrast ratios (minimum 4.5:1) for readability.
    • Text Resizing: Users can resize text up to 200% without loss of functionality.
    • Alternative Text: All images have descriptive alt text for screen readers.
    • Form Labels: All form inputs are properly labeled and associated.
    • Skip Navigation: Skip-to-content links allow quick access to main content.
    • Reduced Motion: Animations respect the prefers-reduced-motion preference.
    • Dark Mode: Optional dark theme for users with light sensitivity.
    • Language Support: Multiple language options available.

    Technical Standards

    This website is built using semantic HTML5 and modern web standards. We regularly test for accessibility issues and make improvements based on user feedback and automated testing tools.

    Known Issues

    While we strive for full accessibility, some third-party services may have limited accessibility features. We're working with vendors to improve their compliance.

    Keyboard Shortcuts

    • Ctrl+K / Cmd+K: Open search overlay
    • Tab: Navigate to next element
    • Shift+Tab: Navigate to previous element
    • Enter: Activate button or link
    • Esc: Close modals and overlays
    • Space: Activate checkbox or button
    • Arrow Keys: Navigate lists and menus

    Assistive Technology Support

    This website has been tested with the following assistive technologies:

    • NVDA (free screen reader)
    • JAWS (premium screen reader)
    • VoiceOver (macOS/iOS)
    • Voice Control (Windows/Mac)
    • Magnification software
    • Voice-to-text applications

    Report Accessibility Issues

    If you encounter accessibility barriers, please report them to us:

    Email: accessibility@government.ge
    Phone: +995 (32) 2 999 999
    Form: Contact us

    Please include specific details about the accessibility issue and the browser/assistive technology you're using. We'll respond within 2 business days.
